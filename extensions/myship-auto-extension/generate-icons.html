MyShip Auto Extension - Icon Generator

Generate orange truck icons for the Chrome extension

16 x 16 Toolbar icon
48 x 48 Extension management
128 x 128 Chrome Web Store

How to use

  1. Click Download on each icon (or Download All).
  2. Place the downloaded files in the icons/ folder of the extension.
  3. The filenames should be: icon16.png, icon48.png, icon128.png.
  4. Reload the extension in chrome://extensions.
